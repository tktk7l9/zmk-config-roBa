# roBa キーマップ チートシート

> Keyball39 配列を roBa（ZMK 分割・右手トラックボール）へ移植した構成。
> 物理は左右 5 列＋親指クラスタ。右手の中央寄り 1 キー（`·`）は内側キー。
> 最終更新: 2026-06-22（英数ホールド → スクロール反映後）

## レイヤー一覧

| # | 名前 | 入り方 | 役割 |
|---|------|--------|------|
| 0 | BASE   | 既定 | 英字・基本入力 |
| 1 | SYMBOL | `Tab` ホールド | 記号 |
| 2 | NUM    | `Space` ホールド | テンキー・矢印・編集 |
| 3 | FUNC   | `Esc` ホールド（右親指） | F キー・BT・リセット |
| 4 | MOUSE  | トラックボール操作で自動 | 自動マウス（カーソル移動） |
| 5 | SCROLL | `英数` ホールド or 内側 `•S` キー | トラックボールでスクロール |

トラックボール: 通常はカーソル移動（レイヤー4 へ自動遷移）／**レイヤー5 が有効な間はスクロール**。

---

## Layer 0 — BASE

```
 Q    W    E    R    T    │    Y    U    I    O    P
 A    S    D    F    G  •S│    H    J    K    L   Ent
 Z    X    C    V    B    │    N    M   M1   M2  ⇧/-
        GUI   ,   .  Tab* Spc* 英数* │ かな* BS          Esc*
```

ホールドタップ（`tap / hold`）:
- `Tab*` = Tab / **SYMBOL(1)**
- `Spc*` = Space / **NUM(2)**
- `英数*` = 英数(LANG2) / **SCROLL(5)** ← 今回変更
- `かな*` = かな(LANG1) / **LCtrl**
- `Esc*` = Esc / **FUNC(3)**
- `⇧/-` = `-` / **LShift**
- `•S` = 内側キー＝押している間 SCROLL(5)（`&mo 5`、押した瞬間からスクロール）
- `M1` `M2` = マウス左/右クリック

## Layer 1 — SYMBOL（Tab ホールド）

```
 %    $    &    ^    /    │    \    {    }    <    >
 @    _    *    +    =    │    _    (    )    ;    :
 `    ~    #    !    ?    │    |    [    ]    '    "
        ─    ─    ─    ─    ─    ─  │  ─    ─           ─
```

---

## Layer 2 — NUM（Space ホールド）

```
 →0   7    8    9    ─    │   Ent  Home  ↑   End  Alt
 0    4    5    6    ─    │   PgUp  ←    ↓    →   PgDn
 ─    1    2    3   Sft   │   ⌥←   ⌥→   M1   M2   M3
        GUI   ─   ─   ─   ─  │ Ctl  Del          FUNC*
```

- `→0` = BASE(0) へ戻る（`&to 0`）
- `⌥←` `⌥→` = Alt+←/→（単語移動）
- `FUNC*` = この位置のホールドで FUNC(3)（`&mo 3`）

---

## Layer 3 — FUNC（Esc ホールド）

```
OUTtg  F7   F8   F9  F10   │   ─    ─    ─    ─   BT0
SysRst F4   F5   F6  F11   │   ─    ─   Caps  ─   BT1
 ─     F1   F2   F3  F12   │   ⌥←   ⌥→   ─   BT2  BTclr
        ─    ─    ─    ─    ─  │ Ctl  ─          Boot
```

- `OUTtg` = 出力切替 USB/BT（`&out OUT_TOG`）
- `SysRst` = ソフトリセット / `Boot` = ブートローダー
- `BT0/1/2` = Bluetooth プロファイル選択 / `BTclr` = ペアリング全消去

---

## Layer 4 — MOUSE（トラックボール操作で自動）

```
 ─    ─    ─    ─    ─    │    ─    ─    ─    ─    ─
 ─    ─    ─    ─    ─    │    ─    ─    ─    ─    ─
 ─    ─    ─    ─    ─    │    ─    ─   M1   M2   M3
        ─    ─    ─    ─    ─    ─  │  ─    ─           ─
```

トラックボールを動かすと自動でこのレイヤーへ。右下に左/右/中クリック。

---

## Layer 5 — SCROLL（英数 ホールド or 内側 •S）

```
 ─    ─    ─    ─    ─    │    ─    ─    ─    ─    ─
 ─    ─    ─    ─    ─    │    ─    ─    ─    ─    ─
 ─    ─    ─    ─    ─    │    ─    ─    ─    ─    ─
        ─    ─    ─    ─    ─    ─  │  ─    ─           ─
```

全キー透過。**有効な間はトラックボールがスクロール**になる。

---

## 凡例

| 表記 | 意味 |
|------|------|
| `─` | 透過（下レイヤーを継承） |
| `*` | ホールドタップ（タップ/ホールドで別動作） |
| `•S` | 内側キー＝モーメンタリ SCROLL |
| `M1/M2/M3` | マウス 左/右/中クリック |
| `⌥←/⌥→` | Alt+矢印（単語移動） |
| `⇧ Sft / Ctl / GUI / Alt` | Shift / Ctrl / Cmd(GUI) / Alt |
| `→0` | レイヤー0 へ固定移動（`&to 0`） |

設定: ホールドタップは `balanced` フレーバー、`lt` は tapping-term 200ms。
ロータリーエンコーダはハード未接続（`disabled`）のため BASE の `inc_dec PG_UP/PG_DN` は非動作。
