# roBa キーマップ チートシート

> Keyball39 配列を roBa（ZMK 分割・右手トラックボール）へ移植した構成。
> 物理は左右 5 列＋親指クラスタ。右手の中央寄り 1 キー（`·`）は内側キー。
> 最終更新: 2026-07-03（roba-hud 自動生成）

## レイヤー一覧

| # | 名前 | 入り方 | 役割 |
|---|------|--------|------|
| 0 | BASE   | 既定 | 英字・基本入力 |
| 1 | SYMBOL | `Tab` ホールド | 記号 |
| 2 | NUM    | `Space` ホールド | テンキー・矢印・編集 |
| 3 | FUNC   | `Esc` ホールド（右親指） | F キー・BT・リセット |
| 4 | MOUSE  | トラックボール操作で自動 | 自動マウス（カーソル移動） |
| 5 | SCROLL | `英数` ホールド or 内側 `•S` キー | トラックボールでスクロール |

トラックボール: 通常はカーソル移動（レイヤー4 へ自動遷移）／**レイヤー5 が有効な間はスクロール**。

## Layer 0 — BASE

```
Q     W     E     R     T    │ Y     U     I     O     P
A     S     D     F     G     ▷5   │ M3    H     J     K     L     ⏎
Z     X     C     V     B     ⌦    │ CW    N     M     M1    M2    -*
⌘     ,     .     ⇥*    ␣*    英数*  │ かな*   ⌫     ⎋*
```

中央 `[ ]` は左右の内側キー（親指/人差し指で届く中央列）。

ホールドタップ（`tap / hold`）:
- `Tab*` = Tab / **SYMBOL(1)**
- `Spc*` = Space / **NUM(2)**
- `英数*` = 英数(LANG2) / **SCROLL(5)**
- `かな*` = かな(LANG1) / **LCtrl**
- `Esc*` = Esc / **FUNC(3)**
- `⇧/-` = `-` / **LShift**

中央・内側キー:
- `[•S]` = 左内側＝押している間 SCROLL(5)（`&mo 5`、押した瞬間からスクロール）
- `[M3]` = 中クリック（`&mkp MB3`）
- `[DEL]` = 前方削除（Delete）
- `[CW]` = Caps Word（次の単語だけ大文字、`&caps_word`）
- `M1` `M2` = マウス左/右クリック

## Layer 1 — SYMBOL（Tab ホールド）

```
%     $     &     ^     /    │ \     {     }     <     >
@     _     *     +     =     ─    │ ─     _     (     )     ;     :
`     ~     #     !     ?     ─    │ ─     |     [     ]     '     "
─     ─     ─     ─     ─     ─    │ ─     ─     ─
```

---

## Layer 2 — NUM（Space ホールド）

```
→0    7     8     9     /    │ ⏎     ↖     ↑     ↘     ⌥
0     4     5     6     ×     −    │ ─     ⇞     ←     ↓     →     ⇟
.     1     2     3     ⇧     +    │ ─     ⌥←    ⌥→    M1    M2    M3
⌘     ─     ─     ─     ─     ─    │ ⌃     ⌦     ▷3
```

- 左ブロックは**テンキー**。演算子 `/ * - + .` を数字の隣に追加（`[-]` `[+]` は内側キー）。
- `→0` = BASE(0) へ戻る（`&to 0`）
- `⌥←` `⌥→` = Alt+←/→（単語移動）
- `FUNC*` = この位置のホールドで FUNC(3)（`&mo 3`）

---

## Layer 3 — FUNC（Esc ホールド）

```
OUT   F7    F8    F9    F10  │ ⏮     ⏯     ⏭     Mute  BTclr*
RST   F4    F5    F6    F11   ─    │ ─     Vol−  Vol+  ⇧⌘4   ☀+    BT1
─     F1    F2    F3    F12   ─    │ ─     ⌥←    ⌥→    ☀−    ⌃⇧⌘4  BT0
─     ─     ─     ─     ─     ─    │ ⌃     ─     BOOT
```

- 右側に**メディア操作**（Prev/Play・Pause/Next/Mute）・**音量**（Vol-/Vol+）・**画面輝度**（Bri-/Bri+）を追加。
- `OUTtg` = 出力切替 USB/BT（`&out OUT_TOG`）
- `SysRst` = ソフトリセット / `Boot` = ブートローダー
- `⌘⇧4` = 範囲スクショ→ファイル（`&kp LG(LS(N4))`・旧Caps位置）
- `⌃⌘⇧4` = 範囲スクショ→クリップボード（`&kp LC(LG(LS(N4)))`・旧BT2位置）
- `BT0/1` = Bluetooth プロファイル選択（BT2は廃止）/ `BTclr` = ペアリング消去（**0.5秒長押しで発動**・タップは無反応＝誤爆防止）

## Layer 5 — SCROLL（英数 ホールド or 内側 •S）

```
─     ─     ─     ─     ─    │ ─     ─     ─     ─     ─
─     ─     ─     ─     ─     ─    │ ─     ─     ─     ─     ─     ─
─     ─     ─     ─     ─     ─    │ ─     ─     ─     ─     ─     ─
─     ─     ─     ─     ─     ─    │ ─     ─     ─
```

全キー透過。**有効な間はトラックボールがスクロール**になる。

## 凡例

| 表記 | 意味 |
|------|------|
| `─` | 透過（下レイヤーを継承） |
| `*` | ホールドタップ（タップ/ホールドで別動作） |
| `•S` | 内側キー＝モーメンタリ SCROLL |
| `M1/M2/M3` | マウス 左/右/中クリック |
| `⌥←/⌥→` | Alt+矢印（単語移動） |
| `⇧ Sft / Ctl / GUI / Alt` | Shift / Ctrl / Cmd(GUI) / Alt |
| `→0` | レイヤー0 へ固定移動（`&to 0`） |

設定: ホールドタップは `balanced` フレーバー、`lt` は tapping-term 200ms。
ロータリーエンコーダはハード未接続（`disabled`）のため BASE の `inc_dec PG_UP/PG_DN` は非動作。
